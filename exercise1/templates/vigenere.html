How it works:

  • The Vigenère cipher uses a keyword to create a repeating key
  • Each letter of the plaintext is shifted by the corresponding letter of the key
  • A = 0, B = 1, ..., Z = 25
  • Ciphertext[i] = (Plaintext[i] + Key[i]) mod 26
